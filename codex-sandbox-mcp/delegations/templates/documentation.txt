# TEMPLATE: JSDoc/TSDoc for Functions

Add JSDoc comments to all exported functions in: {{DIRECTORY}}

For each function, include:
- Brief description (1-2 sentences)
- @param for each parameter with type and description
- @returns with type and description
- @example with realistic usage
- @throws if the function throws errors

Style guide:
- Use present tense ('Returns the...', not 'Return the...')
- Start with a verb
- Keep descriptions concise

Don't modify function bodies, only add comments.
